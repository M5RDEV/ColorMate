# ColorMate - مكتبة ألوان ويب ذكية

مكتبة ألوان ويب ذكية مبنية بتقنيات HTML، CSS، JavaScript النقية مع دعم FontAwesome و jQuery. توفر أدوات شاملة للمطورين والمصممين لاختيار الألوان المناسبة لمشاريعهم.

## التقنيات المستخدمة

- **HTML5** - هيكلة المحتوى
- **CSS3** - التصميم والتخطيط (بدون إطارات عمل)
- **JavaScript (ES6+)** - التفاعل والوظائف
- **jQuery 3.6.0** - تبسيط التفاعلات
- **FontAwesome 6.0.0** - الأيقونات والرموز
- **LocalStorage** - حفظ البيانات محلياً

## كيفية الاستخدام

1. افتح الملف `index.html` في المتصفح
2. استخدم التنقل العلوي للانتقال بين الأدوات المختلفة
3. انقر على أي لون لنسخه إلى الحافظة
4. استخدم أزرار القلب لحفظ العناصر في المفضلة
5. استخدم زر تبديل الوضع المظلم في أعلى الصفحة
